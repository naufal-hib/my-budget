Selamat datang di Dompet Ku
Aplikasi keuangan pribadi yang simpel, clean, dan powerful. Data tersimpan di GitHub kamu — aman & gratis.

Mari setup dalam 4 langkah mudah.
⚙️
Koneksi GitHub
Data kamu akan disimpan sebagai file JSON di repository GitHub. Gratis dan kamu yang kontrol penuh.
Settings → Developer Settings → Personal Access Tokens (Scopes: repo)
ℹ️ Token hanya tersimpan di device kamu (localStorage). Tidak dikirim ke server lain.
🏦
Akun Keuangan
Tambah akun-akun yang kamu miliki. Nama, tipe, dan warnanya bisa diubah kapan saja.
Kategori transaksi akan diisi otomatis dengan preset standar (bisa diubah nanti).
💬
WhatsApp Notifikasi
Terima ringkasan harian, reminder utang, dan alert budget via WhatsApp. Lewati jika belum punya akun Fonnte.
💡 Template pesan, jadwal kirim, dan on/off per jenis notifikasi bisa diatur detail di halaman WhatsApp nanti.
